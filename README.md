# CALC-SHELL.

### Installation

1. `git clone https://github.com/ushanrashmika23/calc-shell.git`

### How to use
1. fist open terminal & run main.py file.

`python main.py`

2. Next select operation.

`+`     add

`-`     reduce

`*`     multiply

`/`     Devide

`?`     History

`$`     Exit

3. Enter fist & second numbers...
